<H1>Tanks game</H1>
<br>

Игра написана на языке Python с помощью библиотеки pygame.
<br>
Управление:
- Игрок 1 - WASD[space]
- Игрок 2 - [↑][←][↓][→]X
<br>

-> При запуске игры мы попадаем в меню, в нем мы можем отключить звук, продолжить игру(PLAY), и перезапустить(RESET)
